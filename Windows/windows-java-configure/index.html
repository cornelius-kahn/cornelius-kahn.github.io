Windows下Java环境的配置

April 3, 2015 Windows PV of this Page

JDK环境变量设置：

  • 新建CLASSPATH：
    .;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;

  • 新建JAVA_HOME：
    C:\Program Files\Java\jdk1.7.0_45
    （JDK安装路径）

  • 编辑Path：
    %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;
    （在原有的参数下的最后添加这一句代码，与前面的用;隔开）
