Git更新代码

June 11, 2016 Git PV of this Page

查看git版本库状态：
git status

获取远端分支到本地临时分支：
git fetch origin master:tmp
查看本地分支：
git branch
比较分支不同：
git diff tmp
Rebase本地临时分支到主分支
git rebase tmp
删除本地临时分支
git branch -d tmp
