html5页面的基本结构

August 16, 2015 Front-end PV of this Page

html5页面的基本结构：

说明：

html5标准网页声明，表示页面采用html5标准。

指定网页语言种类，zh-CN为中文，en为英文。

<head> 标签用于定义文档的头部，它是所有头部元素的容器。

<head> 中的元素可以引用脚本、指示浏览器在哪里找到样式表、提供元信息等等。

文档的头部描述了文档的各种属性和信息，包括文档的标题、在 Web 中的位置以及和其他文档的关系等。

绝大多数文档头部包含的数据都不会真正作为内容显示给读者。

下面这些标签可用在 head 部分：<base>, <link>, <meta>, <script>, <style>, 以及 <title>。

<title> 定义文档的标题，它是 head 部分中唯一必需的元素。例如：

<title>我是萌萌哒的标题</title>

<meta> 元素可提供有关页面的元信息（meta-information），比如针对搜索引擎和更新频度的描述和关键词。

<meta charset=”UTF-8″> 这里指定的是html5的编码集。

<body> 元素定义文档的主体。

<body> 元素包含文档的所有内容（比如文本、超链接、图像、表格和列表等等。）

注意：
<html>、<head>、<title>、<body>均有对应的</html>、</head>、</title>、</body>。它们是成对出现在html页面文档里的。
