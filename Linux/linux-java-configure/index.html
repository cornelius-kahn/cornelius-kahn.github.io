Linux下Java环境的配置

May 7, 2015 Linux PV of this Page

基于RPM软件包的发行版（如RadHat、CentOS、OpenSUSE等）

直接打开.rpm软件包即可

基于dbkg软件包的发行版（如Debian、Ubuntu、Linux Mint等）

Oracle官方没有提供.deb软件包，只能使用.tar.gz软件包

范例中使用的Java安装包是jdk-8u45-linux-x64.tar.gz

  • 进入存放Java安装包的文件夹，解压至相应目录
    sudo tar zxvf jdk-8u45-linux-x64.tar.gz -C /usr/local/java

  • 配置系统变量，打开环境变量配置文件
    sudo gedit /etc/profile

  • 在gedit编辑器中的最后添加如下信息并保存

  • 更新源
    sudo update-alternatives --install /usr/bin/java java /usr/local/java/jdk1.8.0_45/bin/java 300
    sudo update-alternatives --install /usr/bin/javac javac /usr/local/java/jdk1.8.0_45/bin/javac 300
    sudo update-alternatives --config java
